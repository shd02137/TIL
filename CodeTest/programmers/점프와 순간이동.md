# 2018 섬머윈터 코딩 / 점프와 순간이동

## 첫번째 시도

```python
def solution(n):
    ans = 0
    
    while n:
        if n&1==0:
            n=n//2
        else:
            n= n-1
            ans += 1
            
    return ans
```

1. __짝수일경우 n을 반으로 나누고, 홀수일경우 1일빼고 답을 1증가시킨다.__

2. __위의 과정을 n이 0이 될때까지 반복한다.__



문제가 쉬웠다.
