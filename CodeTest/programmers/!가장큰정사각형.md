# 연습문제 / 가장 큰 정사각형

## 첫번째 시도

```python
def solution(board):    
    col = len(board[0])
    row = len(board)
    max_n = min(col,row)
        
    for n in range(max_n,0,-1):
        for i in range(0,col-n+1):
            for j in range(0,row-n+1):
                if board[j][i] == 0:
                    pass
                else:
                    box_sum = 0
                    for n_i in range(0,n):
                        for n_j in range(0,n):
                            if board[j+n_j][i+n_i] == 1:
                                box_sum += 1
                    if box_sum == n*n:
                        return n*n
        
    return 0
```

문제의 풀이는 간단했지만, 효율성 테스트를 통과하지 못해서 고민하고있다.
